# N2 - Dinamismo na WEB

Projeto academico com foco no desenvolvimento de uma interface web para o tema **rede social**, priorizando uma experiencia **intuitiva, moderna e organizada**. A proposta do trabalho foi aplicar conceitos de dinamismo na web por meio da integracao entre estrutura HTML, estilizacao com CSS e comportamentos implementados em JavaScript.

## Sobre o projeto

O site foi idealizado com base no tema **rede social**, resultando em uma interface visualmente atual, com navegacao simples e elementos voltados a interacao do usuario. O trabalho busca demonstrar a construcao de paginas web com organizacao estrutural, identidade visual consistente e recursos dinamicos no front-end.

## Objetivos do trabalho

- Desenvolver uma interface web moderna e funcional.
- Aplicar conceitos de HTML, CSS e JavaScript no mesmo projeto.
- Explorar dinamismo na web por meio de interacoes e manipulacao de elementos da pagina.
- Organizar o projeto de forma clara, facilitando manutencao e entendimento.

## Funcionalidades desenvolvidas

- Interface de rede social com foco em usabilidade.
- Pagina principal com area de publicacao e listagem de atualizacoes.
- Pagina de perfil integrada a navegacao do projeto.
- Aplicacao de logica em JavaScript para dinamismo da interface.
- Manipulacao do DOM para atualizar elementos da pagina de forma interativa.

## Organizacao e desenvolvimento

O projeto foi planejado para unir **design, estrutura e comportamento** em uma proposta coerente de aplicacao web. O Figma foi utilizado como apoio visual no processo de definicao da interface, enquanto a implementacao foi distribuida entre os integrantes conforme suas responsabilidades no desenvolvimento.

## Como executar o projeto

1. Baixe ou clone este repositorio.
2. Abra a pasta do projeto no seu editor.
3. Execute o arquivo [tela_principal.html](./html/tela_principal.html) em um navegador.

## Consideracoes finais

O trabalho **N2 - Dinamismo na WEB** apresenta a construcao de uma interface inspirada em uma rede social, unindo planejamento visual, organizacao estrutural e recursos dinamicos no front-end. O resultado evidencia a aplicacao pratica dos conteudos estudados no desenvolvimento web.
